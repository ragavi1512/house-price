About the application

The application brings together data preparation, a trained pricing model, and an interactive web layer so you can submit property attributes (such as size, location, and age) and receive predicted sale prices along with context you can trust.

Whether you are comparing listings, sanity-checking an offer, or learning how models interpret features, HousePrice is designed to stay transparent: clear inputs, readable outputs, and room to grow into dashboards and saved projects as your workflow matures.

Technology used

Built with a small, dependable stack you can extend as your model and UI evolve.

Python Core language for data handling and model code.
Flask Web framework for routes, templates, and forms.
Jinja2 Server-rendered HTML for fast, accessible pages.
HTML & CSS Semantic layout and responsive styling.
Machine learning Pandas, scikit-learn (or similar) for training and inference.
Deployment-ready WSGI-friendly for production hosts when you go live.
